# 🎓 Student Risk Predictor

This project predicts whether a student is at risk of failing based on:
- Study Hours
- Attendance
- Marks

## 🚀 Technologies Used
- Python
- Pandas
- Scikit-learn

## 📊 Features
- Machine Learning model (Logistic Regression)
- Predicts PASS/FAIL risk
